# SwITCh-Toys
Toy model used by SwITCh project.

# Coding conventions
 - Action and parameter
 ```gaml 
 action my_action(int my_param)
 ```
 - Attribute
 ```gaml
 float my_attribute
 ```
 - Reflex
 ```gaml
 reflex my_reflex
 ```
 - Species
 ```gaml
 species MySpecies
 ```
 - Model
 ```gaml
 model MyModel
 ```
 - Experiment
 ```gaml
 experiment "My experiment"
 ```
 - Display
 ```gaml
 display my_display
 ```
 - GAML folder
 ```
 MyFolder
 ```
  - GAML file
 ```
 MyFile.gaml
 ```
